# 📖 每日讀經

每天按順序讀四章聖經（和合本），電腦、手機、平板都能用的網頁 App。
全書 66 卷 1189 章，每天 4 章，大約 **298 天讀完一輪**，讀完自動開始下一輪。

## ✨ 功能

- **每天四章、按順序**：從創世記讀到啟示錄，App 自動記住你讀到哪
- **經文直接顯示在畫面上**：不用翻聖經、不用開別的 App，點開就能讀
- **進度追蹤**：連續天數 🔥、第幾輪、本輪進度條
- **落後不跳章**：某天沒讀完，隔天從同一個地方繼續，順序永遠不亂
- **離線可用**：讀過的經文會存在裝置上，沒網路也能讀
- **可加入主畫面**：像一般 App 一樣有圖示、全螢幕開啟
- **深色模式**：跟隨系統自動切換
- **字體大小可調**：閱讀頁右上角 A− / A＋

## 🚀 第一次設定（只要做一次）

1. 把這個分支合併到 `main`
2. 到 repo 的 **Settings → Pages**，Source 選 **Deploy from a branch**，
   Branch 選 **main**、資料夾選 **/(root)**，按 Save
3. 等一兩分鐘，你的 App 就會出現在：
   **https://emilywithjesus.github.io/Bible/**

## 📱 加到每台裝置（像裝 App 一樣）

| 裝置 | 做法 |
|---|---|
| iPhone / iPad | 用 Safari 打開網址 → 點「分享」⬆️ → **加入主畫面** |
| Android 手機/平板 | 用 Chrome 打開網址 → 選單 ⋮ → **安裝應用程式**（或「加到主畫面」） |
| 電腦 | 用 Chrome / Edge 打開網址 → 網址列右邊的 **安裝** 圖示 |

## 🔔 讓提醒每天自動跳出來（重點！）

有兩層提醒，建議都開：

### 1. 行事曆提醒（最可靠，每台裝置都會跳通知）

App 裡點 **⚙️ 設定 → 選好時間 → 下載行事曆每日提醒 (.ics)**，
打開下載的檔案加入行事曆。之後每天固定時間，手機、平板、電腦的
行事曆都會跳出「📖 每日讀經時間到了」，點進去就是 App。
（用 Google 日曆或 iCloud 行事曆的話，加一次就會同步到所有裝置。）

### 2. GitHub 每日 Email 提醒（自動附上連結）

這個 repo 內建了排程（`.github/workflows/daily-reminder.yml`）：
合併到 `main` 之後，**每天早上 7:00（台灣時間）** 會自動開一個 Issue，
GitHub 會寄 Email 到你的信箱，點信裡的連結就能開始讀。
讀完把 Issue 關掉，就是今天的打卡 ✅

- 想改時間：編輯 workflow 裡的 `cron`（UTC 時間 = 台灣時間減 8 小時）
- 想關掉：到 repo 的 **Actions** 頁面把這個 workflow 停用

## 💡 小提醒

- 進度存在**各台裝置**上（瀏覽器的 localStorage），不會自動同步。
  換裝置時用 **⚙️ 設定 → 調整進度** 選到目前讀到的卷章即可
- 經文為和合本（公有領域），來源：[thiagobodruk/bible](https://github.com/thiagobodruk/bible)
